## 1. 接口描述

接口请求域名： aiart.tencentcloudapi.com 。

混元生图接口，基于混元大模型，根据输入的文本描述快速生成图片。
默认提供0个并发，代表最多能同时处理1个已提交的任务，上一个任务处理完毕后，才能开始处理下一个任务。

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](/document/api/1668/88071)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | [公共参数](/document/api/1668/88071)，本接口取值：QueryTextToImageJob。 |
| Version | 是 | String | [公共参数](/document/api/1668/88071)，本接口取值：2022-12-29。 |
| Region | 是 | String | [公共参数](/document/api/1668/88071)，详见产品支持的 [地域列表](/document/api/1668/88071#.E5.9C.B0.E5.9F.9F.E5.88.97.E8.A1.A8)。 |
| JobId | 是 | String | 任务 ID。<br/>示例值：1344213737283272704 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| JobStatusCode | String | 当前任务状态码：<br/>1：等待中、2：运行中、4：处理失败、5：处理完成。<br/>示例值：5|
| JobStatusMsg | String | 当前任务状态：排队中、处理中、处理失败或者处理完成。<br/><br/>示例值：处理完成|
| JobErrorCode | String | 任务处理失败错误码。<br/><br/>示例值：ServerError|
| JobErrorMsg | String | 任务处理失败错误信息。<br/><br/>示例值：处理失败|
| ResultImage | Array of String | 生成图 URL 列表，有效期1小时，请及时保存。<br/><br/>示例值：["https://myqcloud.com/result/fd7b5e60-b633-11f0-827b-52540087afa5"]|
| ResultDetails | Array of String | 结果 detail 数组，Success 代表成功。<br/><br/>示例值：["Success"]|
| RevisedPrompt | Array of String | 对应 SubmitTextToImageProJob 接口中 Revise 参数。开启扩写时，返回扩写后的 prompt 文本。 如果关闭扩写，将直接返回原始输入的 prompt。<br/>示例值：["一只金毛"]|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 查询文生图任务

#### 输入示例

```
POST / HTTP/1.1
Host: aiart.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: QueryTextToImageJob
<公共请求参数>

{
    "JobId": "1344213737283272704"
}
```

#### 输出示例

```json
{
    "Response": {
        "JobErrorCode": "",
        "JobErrorMsg": "",
        "JobStatusCode": "5",
        "JobStatusMsg": "处理完成",
        "RequestId": "e4a4eef5-f4aa-40ff-aae0-5e51c5ef5b1e",
        "ResultDetails": [
            "Success"
        ],
        "ResultImage": [
            "https://cos.ap-guangzhou.myqcloud.com/xxx.jpg"
        ],
        "RevisedPrompt": [
            "一只可爱的小狗正蹲在草地上，它的小耳朵竖立着，鼻子微微皱起，显得十分警觉，小狗的毛色是鲜亮的红色"
        ]
    }
}
```

| FailedOperation.ServerError | 服务内部错误。 |
