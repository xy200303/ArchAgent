# ArchAgent UI 设计文档

## 工作台结构

```text
活动栏（主工作区 / 构件库）
├─ 次级面板：场景树或构件库
├─ 中央：R3F 三维视图
└─ 右侧：按需显示的属性检查器与可拖拽对话区域
```

顶部 `SceneToolbar` 负责标题、创建、相机预设、撤销重做和导入导出；各工作区共享该容器，避免维护不一致的头部。

## 视口交互

- 左键拖动空白：平移。
- 滚轮：缩放。
- 拖动右下角导航球：旋转。
- 点击构件：蓝色高亮并同步场景树、检查器。
- 长按选中构件并拖动：预览并放置到吸附地面。
- 绘墙模式：显示明确状态提示，点击起终点后提交命令。

相机动作与构件操作不能争夺同一手势。导航球为 R3F HUD，不增加第二个 WebGL 画布。

## 视觉原则

使用统一 Fluent 令牌、同一图标风格、可见焦点态和不小于 44px 的关键触控热区。构件库使用缓存预览图，不在每张卡片上常驻 WebGL；长列表支持多级滚动与按需加载。
